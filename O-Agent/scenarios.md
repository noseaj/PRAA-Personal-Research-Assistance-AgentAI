# O-Agent 시나리오 분기 가이드

## 개요
사용자의 자연어 질문을 분석하여 적절한 Agent를 선택하는 시나리오입니다.

---

## Agent 소개

### 1. R-Agent (Research Agent)
- **역할**: 논문 검색 및 GitHub 코드 찾기
- **사용 시기**: 
  - 논문 검색이 필요할 때
  - GitHub 레포지토리 찾기
  - 최신 연구 동향 파악
  - SOTA 모델 찾기
- **키워드**: 논문, 검색, 찾아줘, GitHub, 관련 연구, SOTA, 최신, 레퍼런스

### 2. CA-Agent (Code Analysis Agent)
- **역할**: 코드 분석 및 비교
- **사용 시기**:
  - 로컬 코드 분석
  - 구현 방법 설명
  - 코드 구조 파악
  - 알고리즘 설명
- **키워드**: 코드, 분석, 구현, 어떻게, 구조, PyTorch, TensorFlow, 알고리즘

### 3. SA-Agent (Study Assistant Agent)
- **역할**: 연구 일지 분석 및 진행 상황 파악
- **사용 시기**:
  - 연구 일지 분석
  - 진행 상황 요약
  - 다음 단계 추천
  - 연구 방향 제안
- **키워드**: 내 연구, 연구 일지, 진행 상황, 프로젝트, 일지, 로그

---

## 시나리오 유형

### 시나리오 1: R-only
**설명**: 논문 검색만 필요한 경우
**예시**:
- "object detection 관련 논문 찾아줘"
- "Transformer 기반 최신 연구 검색해줘"
- "YOLO GitHub 레포지토리 찾아줘"

**실행**: R-Agent만 실행

---

### 시나리오 2: CA-only
**설명**: 코드 분석만 필요한 경우
**예시**:
- "PyTorch ResNet 구현 분석해줘"
- "이 코드에서 attention mechanism이 어떻게 작동하는지 설명해줘"
- "BERT 모델 구조 설명해줘"

**실행**: CA-Agent만 실행

---

### 시나리오 3: SA-only
**설명**: 연구 일지 분석만 필요한 경우
**예시**:
- "내 연구 일지 분석해줘"
- "최근 진행 상황 요약해줘"
- "내 프로젝트 진행도 알려줘"

**실행**: SA-Agent만 실행

---

### 시나리오 4: SA-then-R
**설명**: 연구 맥락을 먼저 파악한 후 관련 논문 검색
**예시**:
- "내 연구 주제와 관련된 논문 찾아줘"
- "내가 하고 있는 연구의 레퍼런스 검색해줘"
- "내 연구 방향에 맞는 최신 논문 추천해줘"

**실행**: SA-Agent → R-Agent 순차 실행
1. SA-Agent로 연구 맥락 파악
2. 파악된 맥락을 바탕으로 R-Agent 검색

---

### 시나리오 5: R-then-CA
**설명**: 논문을 찾은 후 관련 코드 분석
**예시**:
- "DETR 논문 찾고 구현 코드 분석해줘"
- "Attention is All You Need 논문의 구현 설명해줘"
- "diffusion model 논문 검색하고 코드 비교해줘"

**실행**: R-Agent → CA-Agent 순차 실행
1. R-Agent로 논문 및 GitHub 검색
2. 찾은 코드를 CA-Agent로 분석

---

### 시나리오 6: full-pipeline
**설명**: 연구 파악 → 논문 검색 → 코드 분석 전체 파이프라인
**예시**:
- "내 연구와 관련된 논문 찾고 코드까지 분석해줘"
- "내 프로젝트에 도움될 만한 레퍼런스와 구현 코드 모두 알려줘"

**실행**: SA-Agent → R-Agent → CA-Agent 순차 실행
1. SA-Agent로 연구 맥락 파악
2. R-Agent로 관련 논문 검색
3. CA-Agent로 코드 분석

---

## 분류 가이드라인

### 우선순위
1. **명시적 키워드 우선**: 사용자가 명확히 언급한 Agent 사용
2. **복합 요청**: 여러 Agent가 필요한 경우 순차적 실행
3. **불명확한 경우**: R-only (논문 검색)를 기본값으로

### 판단 기준
- "내" 또는 "우리"가 포함 → SA 고려
- 파일 경로, GitHub URL → CA 고려  
- "검색", "찾아" → R 고려
- "그리고", "그 다음" → 순차 실행 시나리오
